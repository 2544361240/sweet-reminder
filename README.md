# sweet-reminder

给女朋友写的，甜蜜提醒邮件项目，基于大二时写的`java`版本的`girlfriend4j`项目翻译而来，不同的是，以前需要自己写定时挂实验室服务器，现在有了`serverless`，方便多了

# 使用

1. 先开通smtp服务
2. 修改config.yml中的一些配置
3. 部署到腾讯云函数，请选择上传文件夹方式，超时时间设置个30秒应该不过分
4. 设置定时触发器，建议设置晚上某时刻触发，每晚一个甜蜜提示，棒棒的呢
5. enjoy it！！！
